# Limitless App

## About the app
  Limitless app is e-commerce android app that was implemented through Eva Pharma hackathon (Eva Hackathon 2022)
***

## Technologies Used
  * Kotlin Coroutines
  * Dagger Hilt
  * Retrofit
  * MVVM Pattern
  * Solid Principles
  * Google Maps
